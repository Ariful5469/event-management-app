Event-Management-App 

stack : Laravel MVC framework
     Backend-> laravel 
     Frontend-> Tailwind css , Javascript
     Database-> Mysql

IT has 3 types of rules : (use breeze)
      1)Guest
      2)Authenticated User
      3)Admin


1) Guest :  Only access Home Page and see Events.
